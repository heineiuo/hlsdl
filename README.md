# HLSDL

**HLSD**own**L**oad. Download HTTP Live Streaming video as a single file from a url with m3u8 content.

---

## Install

```bash
$ npm i hlsdl -g
```

## Usage

```
$ hlsdl {URL} -o {OUTPUT FILE NAME}
```

## Example

```
$ hlsdl https://video-dev.github.io/streams/x36xhzz/x36xhzz.m3u8 -o x36xhzz.ts
```
